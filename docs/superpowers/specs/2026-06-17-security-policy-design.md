# 安全策略功能完善设计文档

## 概述

将集控管理下的"安全策略"占位页面（SecurityPolicy.tsx）重写为包含 4 个 Tab 的功能页面，覆盖磁盘清理、文件迁移、磁盘格式化、视频音频转文字。

## 范围

**重写：**
- `src/app/components/SecurityPolicy.tsx`（当前为占位，重写为完整页面）

**修改：**
- `src/app/types/permissions.ts` — 补充页面按钮配置

**不受影响：**
- 其他集控管理页面

## Tab 1：磁盘清理

### 功能描述

远程批量清理设备磁盘，支持选择清理类型和自定义文件夹路径。

### UI 布局

```
设备选择区
┌────────────────────────────────────────────────────────────┐
│  目标设备：[下拉选择器] [添加设备]  已选 3 台               │
│  已选：东教学楼101、东教学楼102、西教学楼201               │
└────────────────────────────────────────────────────────────┘

磁盘选择与清理配置
┌────────────────────────────────────────────────────────────┐
│  目标磁盘： [C:\ ▼]  [D:\ ▼]  [E:\ ▼]                    │
│                                                           │
│  ☑ 系统盘清理项目：                                        │
│  ☑ 清理备份文件          预计释放：约 2.3 GB               │
│  ☑ 清理缓存文件          预计释放：约 1.1 GB               │
│  ☑ 清理日志文件          预计释放：约 512 MB               │
│                                                           │
│  ── 自定义文件夹 ──                                        │
│  [C:\Temp] [浏览]  [➕ 添加路径]                           │

操作区
┌────────────────────────────────────────────────────────────┐

┌────────────────────────────────────────────────────────────┐

│  东教学楼101  120GB   118.5GB  1.5GB   ✅ 成功            │
│  东教学楼102  128GB   126.2GB  1.8GB   ✅ 成功            │
│  西教学楼201  120GB   118.0GB  2.0GB   ❌ 失败（磁盘忙）   │

```

### 交互逻辑

- 设备选择：多选设备，支持全选当前 Tab 中的设备
- 磁盘选择：每个设备可能有多个盘符（C:、D:），选择要清理的磁盘
- 清理项目：复选框选择（备份/缓存/日志），每项显示预计释放空间（mock）
- 自定义文件夹：动态添加/删除路径，每个路径可单独浏览选择
- 预览：模拟计算可释放空间，不实际执行
- 执行：触发远程清理，实时显示进度
- 结果：表格展示每台设备的执行结果

- 预览显示待格式化设备列表
- 执行前弹出二次确认对话框

```
│  └────────────────────────────────────────────────────┘   │
│                                                           │

│  │                                                    │   │

│  已标记删除片段：3 段  [🗑️ 确认删除片段]  [💾 导出文字]  │
└────────────────────────────────────────────────────────────┘
```

### 交互逻辑

- 选择视频文件后可在播放器中预览
- 点击「提取音频并转文字」模拟转写过程（mock 数据）
- 转写结果按时间分片展示，每行显示时间段和文字内容
- 每行文字可编辑（点击直接修改文本）
- 每行文字有删除按钮，删除时对应视频片段被标记
- 底部「确认删除片段」执行实际片段移除操作
- 「导出文字」将转写结果导出为文本文件

## 类型定义

```typescript
// Tab 通用
type SecurityTab = 'disk-cleanup' | 'file-migration' | 'disk-format' | 'audio-transcribe';

interface DeviceAction {
  deviceId: string;
  deviceName: string;
  status: 'pending' | 'running' | 'success' | 'failed';
  detail?: string;
}

// 磁盘清理
interface CleanupConfig {
  disk: string;           // 盘符如 C:/
  cleanBackup: boolean;
  cleanCache: boolean;
  cleanLog: boolean;
  customPaths: string[];  // 自定义文件夹路径
}

interface CleanupResult extends DeviceAction {
  beforeSpace?: string;
  afterSpace?: string;
  freedSpace?: string;
}

// 文件迁移
interface MigrationConfig {
  sourceDisk: string;
  targetDisk: string;
  migrateVideo: boolean;
  migrateImage: boolean;
  migrateMusic: boolean;
  migrateDocument: boolean;
  createShortcut: boolean;
  onlyOldFiles: boolean;
}

interface MigrationResult extends DeviceAction {
  fileCount?: number;
  dataSize?: string;
}

// 磁盘格式化
interface FormatConfig {
  targetDisk: string;
  fileSystem: 'NTFS' | 'FAT32' | 'exFAT';
  volumeLabel: string;
}

// 音频转文字
interface TranscriptSegment {
  id: string;
  startTime: number;
  endTime: number;
  text: string;
  deleted: boolean;
}

interface TranscriptionState {
  status: 'idle' | 'processing' | 'done' | 'error';
  segments: TranscriptSegment[];
  videoName?: string;
}
```

## 技术实现

### 组件结构

```
SecurityPolicy.tsx
├── Tab 切换（4 个 Tab）
├── DeviceSelector（设备选择器，跨 Tab 复用）
├── Tab1: DiskCleanupPanel
│   ├── 磁盘/清理项配置
│   ├── 自定义文件夹列表
│   └── 结果表格
├── Tab2: FileMigrationPanel
│   ├── 源/目标磁盘选择
│   ├── 文件类型配置
│   └── 结果表格
├── Tab3: DiskFormatPanel
│   ├── 磁盘选择（过滤系统盘）
│   ├── 双重确认复选框
│   └── 结果表格
└── Tab4: AudioTranscribePanel
    ├── 视频选择/播放器
    ├── 转写操作按钮
    ├── 转写结果列表（可编辑/删除）
    └── 删除片段确认
```

### 依赖

无新增依赖。使用现有 MUI 组件。
