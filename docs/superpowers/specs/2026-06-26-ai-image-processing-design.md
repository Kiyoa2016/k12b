# AI 图片处理功能设计文档

## 概述

在"果仁云菜单"下新增"AI 模块"分组，其下新增"AI 图片处理"页面。支持上传图片后选择 AI 功能进行处理：图片优化、AI 重新生成、智能抠图、提取文字。

## 范围

**新增文件：**

**修改文件：**

- `src/app/types/permissions.ts` — 添加页面和按钮配置

## 交互流程

1. **上传图片** — 拖拽或点击上传，显示原图预览
2. **选择功能** — 4 个功能 Chip 单选，选中后显示对应参数
3. **配置参数** — 强度、风格、背景等（见下方）
4. **执行** — 点击"开始处理"，显示加载动画，2-3 秒后展示结果
5. **结果操作** — 下载结果图、复制文字、重新上传

## 四个功能参数详情

### 1. 图片优化
- 参数：优化强度 — Chip 选择（低 / 中 / 高）
- Mock 效果：模拟增强画质，显示"优化后"图片（加 CSS 滤镜模拟）

### 2. AI 重新生成
- 参数：风格 — Chip 选择（写实 / 插画 / 油画 / 素描）
- Mock 效果：显示模拟生成的图片

### 3. 智能抠图
- 参数：替换背景（开关）、背景颜色（颜色选择器）
- Mock 效果：显示去除背景后的透明 PNG 效果（棋盘格背景），如有替换背景则叠加背景色

### 4. 提取文字
- 参数：无
- Mock 效果：返回一段模拟的 OCR 识别文字，支持一键复制

## 状态管理

```
无图片 → 展示上传区域，功能按钮禁用
已上传 → 显示预览，功能可选
处理中 → 按钮加载态，禁用所有操作
完成   → 显示结果
错误   → 显示错误提示，允许重试
```
